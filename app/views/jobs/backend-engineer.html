Backend Engineer

Our Mission


College is about meeting new people and sharing unforgettable experiences together. We help students create these long-lasting memories by showing them events both on and off-campus that we think they’ll love.



The Challenge


UEvents strives to provide users with personalized event recommendations that provide them with a plethora of exciting options to tap into at any given time. As we expand into more places and take on more users and events, we need to maintain a scalable architecture that successfully stores and manipulates the data we collect.


So how will we ensure that when a user opens up our platform we are able to provide them with a stellar experience - to provide them with great event recommendations in a heartbeat?



The Role


Design and build new features from the ground up: Crafting UIs with front-end engineers and building backend systems with Ruby


Practice disciplined software engineering, including automated testing, code reviews, and writing beautiful, pragmatic code.


Apply user testing to collect feedback/metrics and rapidly iterate on product


Be flexible, grow, and learn fast in a startup environment



About You


You’re as enthused as we are about fundamentally changing local event discovery and you can't wait to get started.


You have a passion for working on live projects that impact real people. We care more about experience and enthusiasm than degrees and awards


Familiarity with Git, MVC frameworks and a willingness to learn new technologies. Our technology is built on a Ruby on Rails backend with an AngularJS frontend. You should be prepared to learn what you need to get the job done.


You are a self-starter. You can take a project and run with it, but also know to ask for help when you’re stumped.


Quality and cleanliness of code matters to you. You understand that taking shortcuts now don't pay off in the long run.
